### 🍒 Shoppinng List (React)
`📁 fruits-catalogue`

**Description**  
A small React component that demonstrates searching and selecting items in a list.

**Key features**

- Controlled input for search
- Checkbox selection stored in state
- `useMemo` to memoize filtered results based on the search query
- `useCallback` to memoize the toggle handler (useful when passing callbacks to memoized child components)

**How to run**
```bash
cd fruits-catalogue/my-fruits
npm install
npm run dev
```
